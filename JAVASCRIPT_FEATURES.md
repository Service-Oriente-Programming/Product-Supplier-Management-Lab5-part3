## 🛠️ **API Endpoints**

### Search API
```javascript
GET /api/search?q=searchTerm&supplier=supplierId
```
Returns JSON with search results.

### Suppliers API
```javascript
GET /api/suppliers
```
Returns all suppliers for dropdown population.

### Statistics API
```javascript
GET /api/stats
```
Returns application statistics (total products, suppliers, value, etc.).
